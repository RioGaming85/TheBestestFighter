# TurnBasedFightingGame
A basic 1 on 1 fighter, but turn based.

### Features:
 - Light attack
 - Heavy attack
 - Block
 - Move with recoil, __*pkmn*__ style
 - Healing *(je suis Genji)*
 - Custom moves for each class
 - The Ultimate move
